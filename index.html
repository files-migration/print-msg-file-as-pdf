How to Print MSG file as PDF with Outlook directly

  • Select your MSG file and open it with Outlook on your computer’ screen.
  • Now go to File Menu and choose Print option
  • After that choose Microsoft Print to PDF option.
  • If you want to print MSG file as PDF with attachments, then you can go with Print-options from Microsoft Print to PDF option and tick on the save emails to PDF with attachments option. But for that make sure that your printer must set as default printer otherwise it will not print MSG file as PDF with attachments.
  • Click on OK button and choose location to save email in PDF.
